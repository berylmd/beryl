Expriments for a simpler beryl archetecture.

- pnpm
- turborepo
- electron


---

# Running

## Prereqs

- `pnpm`

This project uses pnpm workspaces, so have pnpm installed globally.

## Deps

```
# in /beryl-red dir
pnpm i
```

## Development

```
pnpm run dev
```

will spin up 2 dev servers and launch the electron app.

## Prod Build

```
pnpm run build
```
